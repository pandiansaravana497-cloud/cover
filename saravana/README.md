# Book Front Cover Page Design

## Ex 05 - Book Front Cover Page Design

A responsive book front cover page designed using **HTML and CSS**.

## Project Structure

```
ragul/
└── ragul/
    └── static/
        └── book.html   ← Main cover page
README.md
image.png               ← Author photo (embedded in HTML)
```

## Features

- Sky blue gradient background with decorative floral SVG pattern
- Bold book title: THE PATH TO WISDOM
- Italic subtitle: "Navigating Life's Challenges with Insight and Clarity"
- Author photo embedded directly in HTML (no external file needed)
- Dark navy footer bar with author name and year/college code
- Book spine effect using CSS

## How to Run

1. Clone or download this repository
2. Open `ragul/ragul/static/book.html` in any browser — no server needed

## Live Demo

Open `book.html` directly in your browser (File Explorer → double click)
